【JSON テンプレート】
[
  {
    "term": "（問題にしたい用語や作品名など）",
    "aliases": ["別名や読み方", "他の表記"],
    "hint": "答えを思い出すヒント。特徴的なキーワードや連想しやすい要素を簡潔に書く。",
    "background": "成立した時代背景・社会的文脈・政治や経済との関わりを説明する。",
    "impact": "後世に与えた影響。思想や美術・信仰・制度の発展への波及など。",
    "academic_value": "学問的意義。この史料や作品がなぜ研究対象として重要なのか。大学入試で問われやすい理由。",
    "comparison": {
      "previous": "前の時代との比較。特徴の違いや連続性を説明する。",
      "contemporary": "同時代の他の思想・文化との比較。位置づけや対照を示す。",
      "next": "次の時代への影響や変化の比較。どのように発展・変容したか。"
    },
    "image": {
      "commonsTitle": "File:ファイル名.jpg",
      "note": "画像の簡単な説明（例：○○寺本堂、△△像など）"
    }
  }
]

書き方のポイント

term: 正解として入力させたい語（例：「運慶」「『歎異抄』」）

aliases: 同じ意味で答えと認めたい表記（例：「うんけい」「歎異抄」など）

hint: 単なる暗記ではなく、答えに結びつくイメージワードを並べる

background / impact / academic_value: それぞれ独立した短い文章（用語名は本文に入れない）

comparison: 「前代／同時代／次代」で三方向の比較を必ず入れる

image.commonsTitle: Wikimedia Commons のファイル名を指定（File: から始まる）

image.note: その画像が何なのかの短い説明
